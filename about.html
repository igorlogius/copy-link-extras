Adds a link context menu quickly copy a link text, a link text with url and also allows copying the url with and without parameters. Usage
  1. install the addon
  2. visit a non website
  3. right click on a link
  4. in the context menu hover over the "Copy Link Extras"
  5. in the sub context menu click on "Text", "Text + URL", "URL (without Parameters)" or "Text + URL (without Parameters)"
  6. now the content should be in the clipboard and can be pasted
